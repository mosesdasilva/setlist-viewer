# Mission: Maintain Setlist Viewer confidently

## Why
Learn enough of this repository to run the viewer, find the right file, make small safe edits, verify them, and understand how changes reach GitHub Pages.

## Success looks like
- Explain the roles of HTML, CSS, JavaScript, chart sources, generated files, tests, and deployment
- Run both the split viewer and portable viewer locally
- Choose the correct source file for a content, appearance, or behavior change
- Rebuild and test without editing generated output

## Constraints
- Beginner with little or no programming experience
- Learn through a visual HTML guide and small practical steps
- Preserve the static, zero-package project design

## Out of scope
- Mastering HTML, CSS, JavaScript, Python, or Git in one lesson
- Redesigning or changing the viewer
